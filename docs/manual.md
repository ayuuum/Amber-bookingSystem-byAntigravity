## 1. 管理者ログイン
1. ログインページ（`/login`）にアクセスします。
2. 登録済みのメールアドレスとパスワードを入力し、「ログイン」ボタンをクリックします。
3. 認証に成功すると、管理画面（ダッシュボード）に遷移します。

## 2. 管理画面（ダッシュボード）
ダッシュボードでは、システム全体の状況を一目で確認できます。
- **今日の予約**: 本日の予約件数と詳細が表示されます。
- **売上概況**: 今月の売上や前月比などを確認できます。
- **クイックアクション**: 新規予約作成や設定画面へのショートカットがあります。

## 3. 予約管理
### 予約の確認
サイドメニューの「カレンダー」または「予約一覧」から、全ての予約を確認できます。
- **カレンダー表示**: 月/週/日単位で予約状況を視覚的に確認できます。
- **詳細確認**: 予約をクリックすると、顧客情報やサービス内容、金額などの詳細が表示されます。

### 予約の手動作成
電話などで受けた予約を手動で登録する場合：
1. 「新規予約作成」ボタンをクリックします。
2. 顧客情報、希望日時、サービスを選択します。
3. 「作成」を押すと予約が確定し、カレンダーに反映されます。

## 4. 設定変更
### プロフィール設定
右上のアイコンメニューから「プロフィール」を選択します。
- **名前更新**: 表示名の変更が可能です。
- **パスワード変更**: セキュリティのため、定期的な変更を推奨します。

### 店舗情報（管理者のみ）
「店舗管理」メニューから、営業時間や提供サービスの編集が行えます。
